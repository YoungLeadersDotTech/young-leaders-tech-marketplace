# Changelog

All notable changes to the eval-committee plugin.

## [1.0.0] - 2026-06-14

### Added
- Initial public scaffold of eval-committee, generalised from a private eval suite.
- Generic committee of four expert personas with default weights.
- Phase model (discovery, score, synthesize), default rubric, and weighted verdict.
- Opt-in presets hook so private benchmarks and criteria stay in a private repo.
- Eval template and a synthetic example.
